# 钓一盏鱼灯 v3 — 粒子点云交互系统

> 非遗鱼灯 × 手势交互 × 实时粒子 × 沉浸式体验

## 项目概述

基于 Three.js 粒子点云的互动艺术装置网页端，用户通过摄像头手势、麦克风吹气、手机遥控等方式，体验从"水面观鱼 → 钓鱼 → 制灯 → 放生"的完整非遗鱼灯工艺故事。

## 运行方式

```bash
# 在 web/v3 目录下启动本地服务器
cd H:\ai_nbb\web\v3
python -m http.server 8081

# 浏览器打开（建议 Chrome）
http://localhost:8081/

# 如需手机连接，用局域网 IP
http://<你的IP>:8081/
```

**要求**：Chrome 90+ / Edge 90+ / Safari 15+（需支持 WebGL2、MediaPipe、Web Audio）

## 交互流程（四幕结构）

```
┌──────────┐    捏合/2.5s    ┌──────────┐    捏合    ┌──────────┐  仪式手势   ┌──────────┐
│  水面    │ ──────────────→ │  钓鱼    │ ────────→ │  制灯    │ ──────→ │  放生    │
│  WATER   │                 │  FISHING │           │ CRAFTING │         │ RELEASE  │
└──────────┘                 └──────────┘           └──────────┘         └──────────┘
     ↑                                                                        │
     └────────────────────── 张手上抬（循环） ────────────────────────────────┘
```

**制灯阶段形态顺序**（对应非遗工艺）：
```
活鱼(参考) → 骨架(扎竹) → 糊纸(蒙皮) → 上色(彩绘) → 鱼灯(成品) → 🐟鱼群(放生)
```

| 阶段 | 视觉 | 交互 |
|------|------|------|
| **水面** | 物理水波纹（FBO ping-pong） | 👆指尖触碰=波纹，🤏捏合=进入钓鱼 |
| **钓鱼** | 鱼影从水下缓慢浮出，暗蓝色调 | 手掌吸引鱼影靠近，🤏捏合=钓起 |
| **制灯** | 粒子点云鱼灯（5种工艺形态） | 双手推开、抹过、握拳张开、张手靠拢上托、张手上抬 |
| **放生** | 600条鱼群（InstancedMesh Boids） | ✊拳头=鱼群追随，🖐️张手=鱼群散开 |

## 手势系统

基于 MediaPipe Hands 实时检测，6种互不冲突的手势：

| 手势 | 检测原理 | 触发条件 | 功能 |
|------|---------|---------|------|
| 👆 指尖触碰 | 5指尖位置传入 FBO | 始终有效 | 水波纹 + 引鱼 |
| 🤏 捏合 | 拇指-食指距离 < 0.04 | 60帧冷却 | 阶段切换（钓起/抓住/回到水面） |
| ↔ 双手推开 | 双手腕 X 距离持续扩大 | stage0 + 90帧冷却 | 活鱼 → 骨架 |
| ☞ 单手抹过 | 手腕从左侧向右累计移动 | stage1 + 90帧冷却 | 骨架 → 糊纸 |
| ✊→🖐️ 握拳张开 | fingersUp 从 ≤1 到 ≥4 | stage2 + 90帧冷却 | 糊纸 → 上色 |
| 🙌 张手靠拢上托 | 双手张开，从两侧靠近并整体上移 | stage3 + 90帧冷却 | 上色 → 鱼灯 |
| 🖐️ 张手上抬 | 张手保持后手腕上移 | stage4/放生 + 90帧冷却 | 放生回水面 |
| 🖐️ 张手 | fingersUp ≥ 4，**保持0.5s** | 与仪式冷却共同防误触 | 粒子整体爆炸式散开 / 鱼群驱散 |
| ✊ 握拳 | fingersUp ≤ 1 | — | 粒子聚拢 / 鱼群追随拳头 / 缩放 |

**防冲突设计（三重保护）**：
- 仪式手势：必须按工艺顺序推进，并有全局冷却防止连跳
- 散开：需张手保持0.5s（瞬间张开不触发）
- 缩放：仅握拳时Y映射到相机Z
- 跟随：始终有效（柔和插值0.04），不受手型限制

## 配置说明

`app.js` 顶部 CONFIG 对象：

```js
const CONFIG = {
    particleGrid: 256,    // 粒子网格尺寸（256²=65536）
    pointSize: 2.5,       // 粒子大小
    relief: 0.0,          // 默认浮雕深度（0=平面）
    cameraZ: 160,         // 相机初始距离
    morphDuration: 1.5,   // 变形过渡秒数
    boidsCount: 600,      // 鱼群数量
};
```

## 已知限制

- MediaPipe 需要 HTTPS 或 localhost 才能获取摄像头
- 麦克风吹气检测对环境噪声敏感（安静环境效果最佳）
- PeerJS 依赖公网信令服务器（`0.peerjs.com`），纯内网无法使用
- 手机连接要求同一局域网且用 IP 地址访问（非 localhost）
- 鱼群 Boids 在低端设备可能掉帧（可降低 boidsCount）
